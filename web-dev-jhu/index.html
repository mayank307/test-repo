Description

This course offered by coursera in connection with John Hopkins University. The course title is Html, Css and JavaScript for Web Developers, and the instructor for this course is Yaakov Chaikin.

The course takes us from basics to intermediate levels of web development. Throughout the course we will learn how Html gives Structure to web pages, while Css is used to Style it and JavaScript is used to provide Behavior or Functionality and make the Web more dynamic in nature.
